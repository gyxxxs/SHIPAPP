# 船舶电气系统维护检查清单

## 日常检查（每日）

### 主配电板
- [ ] 检查指示灯是否正常
- [ ] 检查仪表读数是否正常
- [ ] 检查运行声音是否正常
- [ ] 检查是否有异常气味
- [ ] 记录运行参数

### 关键设备
- [ ] 检查电机运行状态
- [ ] 检查保护装置状态
- [ ] 检查监测系统状态
- [ ] 检查通信链路状态

---

## 周检（每周）

### 连接检查
- [ ] 检查主配电板连接紧固度
- [ ] 检查关键回路连接
- [ ] 检查接地系统连接
- [ ] 记录检查结果

### 外观检查
- [ ] 检查设备外观是否正常
- [ ] 检查是否有异常发热
- [ ] 检查是否有异常振动
- [ ] 清洁设备表面

---

## 月检（每月）

### 绝缘测试
- [ ] 测试主配电板绝缘电阻（标准：≥0.5MΩ）
- [ ] 测试关键回路绝缘电阻
- [ ] 测试电机绝缘电阻
- [ ] 记录测试结果

### 接地系统检查
- [ ] 测试系统接地电阻（标准：≤4Ω）
- [ ] 测试设备接地电阻（标准：≤10Ω）
- [ ] 检查接地连接
- [ ] 记录测试结果

### 功能测试
- [ ] 测试保护装置功能
- [ ] 测试监测系统功能
- [ ] 测试通信功能
- [ ] 记录测试结果

---

- [ ] 测试设备性能参数

- [ ] 记录测试结果

- [ ] 更换老化部件

- [ ] 更新维护记录

## 检查记录要求

### 必须记录的内容
1. 检查日期和时间
2. 检查人员姓名和资质
3. 检查内容和结果
4. 发现的问题和处理措施
5. 使用的工具和材料
6. 下次检查计划
7. 检查负责人签字

### 记录保存
- 维护记录至少保存3年
- 重要记录永久保存
- 电子记录定期备份
- 纸质记录妥善保管

---

## 异常情况处理

### 发现异常时
1. 立即记录异常情况
2. 评估异常严重程度
3. 采取相应处理措施
4. 通知相关人员
5. 跟踪处理结果

### 紧急情况
1. 立即断电，确保安全
2. 隔离故障设备
3. 启动应急程序
4. 通知岸基支持
5. 详细记录过程
